# BDWP-TS

一个基于 Bun 的全栈 TypeScript Monorepo，包含后端服务与 Vue 前端。

## 快速开始

```bash
# 安装依赖
bun install

# 同时启动后端和前端
bun run dev

# 构建
bun run build
```

## 项目结构

- `apps/backend`: Bun + Elysia + Drizzle
- `apps/frontend-vue`: Vue 3 + Vite + Pinia
